# Shape Up Interactive Book Guide

A single-page interactive study app based on the book *Shape Up* by Ryan Singer (Basecamp).

## Overview

This project is a lightweight, no-build web app that runs directly in the browser.

It includes:

- Structured content for the three core parts of Shape Up: Shaping, Betting, and Building.
- Chapter-by-chapter reading with key takeaways.
- Progress tracking for completed chapters.
- Part quizzes with score feedback.
- Glossary of key Shape Up terms.
- Interactive Hill Chart visualization (drag-and-drop dots).
- Shape Up cycle visualization.

## How to Run Locally

### Option 1: Open directly

1. Open `index.html` in your browser.

### Option 2: Use VS Code Live Server

1. Open the project in VS Code.
2. Start Live Server on `index.html`.
3. Open the provided local URL.

## Deployment (GitHub Pages)

1. Push the repository to GitHub.
2. In GitHub, go to **Settings > Pages**.
3. Under **Build and deployment**, set source to deploy from your default branch.
4. Save and wait for GitHub Pages to publish.
5. Copy the generated site URL and replace the placeholder in the **Live Demo** section above.
